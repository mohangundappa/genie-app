# Data Genie — AI-Powered Data Assistant

A Databricks Genie-like system that lets you ask questions about your data in natural language. Data Genie translates your questions into SQL queries, executes them, and visualizes the results with interactive charts.

![Data Genie Screenshot](docs/screenshot.png)

## Features

- **Natural Language to SQL** — Ask questions in plain English; get accurate SQL queries generated by GPT-4o or a built-in fallback engine
- **Interactive Visualizations** — Auto-suggested bar, line, pie, area, and scatter charts via Recharts
- **Dataset Explorer** — Browse loaded datasets, view schemas, column types, and row counts
- **Query History** — Review and re-run past queries
- **SQL Transparency** — See the generated SQL with syntax highlighting and one-click copy
- **Configurable LLM** — Choose between GPT-4o, GPT-4o Mini, GPT-4 Turbo, or GPT-3.5 Turbo
- **Works without API key** — Built-in pattern matcher handles common queries when no OpenAI key is configured

## Quick Start

### Backend

```bash
cd genie-backend
poetry install
poetry run fastapi dev app/main.py --port 8000
```

### Frontend

```bash
cd genie-frontend
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

### Configure OpenAI (Optional)

Click **Settings** in the sidebar to enter your OpenAI API key, or create a `.env` file in `genie-backend/`:

```
OPENAI_API_KEY=sk-your-key-here
```
